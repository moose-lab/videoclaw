# 故事类分镜模板

## 模板结构

【素材介绍】：
【图1】<角色/物体描述>
【图2】<角色/物体描述>

【场景描写】：<整体背景描述，可选>

【背景音】：<整体背景音描述，可选>

【分镜】：
【镜头标题】（X-Y秒）
画面：<包含动作、台词、情绪的完整描述>
音效：<此镜头的音效>

【注意事项】：
- 角色设定：<关键角色特征>
- 氛围：<整体氛围要求>
- 动作节奏：<节奏要求>
- 细节还原：<关键细节>

## 使用说明

此模板采用"总分总"结构：

1. **总（素材介绍）**：列出所有素材（角色/物体），确保每个元素在首次出现时有完整描述
2. **分（分镜）**：逐镜头描述画面、动作、台词和音效
3. **总（注意事项）**：汇总全局约束（角色设定、氛围、节奏、细节）

### 音效标注规范

- 每个镜头应标注关键音效（门声、脚步、环境音等）
- 音效服务于叙事节奏：紧张场景用尖锐/突发音效，温柔场景用环境白噪音
- 音效与画面同步，标注在对应镜头的音效字段中
- 无音效的镜头留空即可

### 素材描述规范

- 角色在首次出现时必须包含完整外貌描述（年龄、性别、体型、发型、服装、标志性特征）
- 物体需描述材质、颜色、大小等关键视觉特征
- 后续镜头可简化为角色名，但视觉提示词（visual_prompt）中仍需保持完整描述
